# SOP-注塑机-料筒清洁标准作业指导书

- **设备型号**：海天 MA2500/1000（Haitian MA Series）
- **适用场景**：换料前清洁、长期停机封机、出现黑点/拉丝/分层
- **预计工时**：1.5 小时
- **风险等级**：中（高温熔体）

## 一、安全前置

1. 关闭料斗，将剩余原料抽干净，避免混料污染。
2. 确认机筒温度处于工作温度（避免冷启动剪切螺杆）。
3. 备好清洗料（PP 或专用清洗剂 Asaclean U），数量 = 螺杆容量 × 3。
4. 防护：耐高温手套、面罩、长袖工作服。

## 二、清洗步骤

1. 将清洗料加入料斗，连续注射空打，直至射出料色稳定。
2. 若黑点仍多：**降温清洗法**——逐段降低 20℃，每段空打 5 模，让残留焦料随冷流挤出。
3. 切换到目标新料前，再用新料过渡 10–15 模。
4. 检查首件外观：透光性、色差、流痕，确认合格后正式投产。

## 三、判定标准

| 指标 | 合格 | 警戒 | 不合格 |
|---|---|---|---|
| 黑点（每 dm²） | 0–1 | 2–3 | ≥ 4 |
| 色差 ΔE | ≤ 1.0 | 1.0–2.0 | > 2.0 |
| 拉丝长度 | 无 | < 5 mm | ≥ 5 mm |

## 四、深度清洁（季度执行）

1. 拆下料筒喷嘴，使用铜刷与专用通条清理流道。
2. 螺杆拆出后，用紫铜刮刀刮除附着层，**严禁使用钢刷**。
3. 检查止逆环密封性：闭口压力测试，泄漏率 ≤ 5%。
4. 装回扭矩按厂家手册（喷嘴 350 N·m，螺杆中段 280 N·m）。

## 五、典型故障关联

- 频繁黑点 → 检查料斗干燥温度与物料含水率。
- 色差超标 → 优先怀疑止逆环磨损（残料留在止逆腔）。
- 拉丝 → 喷嘴温度过高或保压切换点过晚。
