# SOP-CNC-轴承更换标准作业指导书

- **设备型号**：CNC-650 立式加工中心
- **适用故障**：主轴异响、振动值 > 4.5 mm/s、温升超过 15℃/小时
- **预计工时**：3.5 小时（双人协作）
- **风险等级**：中（涉及主轴拆装）

## 一、安全前置

1. 切断主电源并挂"禁止合闸"标识牌（LOTO）。
2. 等待主轴自然冷却至 35℃ 以下，避免烫伤与热变形测量误差。
3. 准备工装：液压拉马、扭矩扳手（0–200 N·m）、千分表、专用压装套筒。
4. 防护用品：防割手套、护目镜。

## 二、拆卸步骤

1. 拆下主轴防护罩，记录定位销位置。
2. 使用千分表记录主轴轴向窜动与径向跳动**初始值**（用于复装后比对）。
3. 卸下锁紧螺母，使用液压拉马匀速拉出旧轴承，**严禁敲击**。
4. 清洁轴承位，检查轴肩是否有拉伤；若拉伤深度 > 0.02 mm，需返修轴肩。

## 三、装配步骤

1. 选用 **NSK 7014C P4 角接触轴承**，配对方式 DB（背靠背）。
2. 加热轴承至 80–90℃（感应加热器，禁用明火），快速套入轴肩。
3. 锁紧螺母按 **80 N·m → 120 N·m → 150 N·m** 三段式分步施加扭矩。
4. 涂抹高速主轴脂（Klüber Isoflex NBU 15），填充量 30%–35%，**禁止过满**。

## 四、复装与验证

1. 复装防护罩，按拆卸时的定位销标记装回。
2. 启动空载试运行：3000 rpm × 5 min → 6000 rpm × 5 min → 10000 rpm × 10 min。
3. 振动值需 ≤ 2.5 mm/s，温升需 ≤ 8℃/小时；超标则停机复查预紧力。
4. 完成后在 CMMS 中关闭工单，附录振动与温度曲线截图。

## 六、常见错误

- 用铜棒敲击轴承内圈 → 必现保持架变形，三个月内必坏。
- 预紧力过大 → 高速段温升不收敛。
- 主轴脂填充过满 → 启动时甩脂污染编码器。
